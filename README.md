# Navvix Limited landing page

A lightweight static page for Tim Norris-Wiles (Navvix Limited). Deployable on GitHub Pages for easy domain pointing.

## Deploying to GitHub Pages
1. Push the repository to GitHub.
2. In the repository settings, enable **GitHub Pages** (Pages). Choose the **main** branch and root directory.
3. If using a custom domain, add it in the Pages settings and update your DNS with the provided A/AAAA or CNAME records.

## Local preview
Simply open `index.html` in your browser.
